Icons

  • View Documentation

Icons are used to clearly and visually represent some information at-a-glance. They may be used alone (without accompanying text) or in addition to label text. We've selected a well-rounded set of icons along with a few variations, however the set is customizable per your institutions' needs.

Using Icons (Decorative and as part of controls/content)

We've organized the icon fonts that edX uses from the Font Awesome 4.6.3 library. However the full library of icons is available.

Icons can be used as both visual decorators or as tangible parts of a UI's controls and content. In the latter case, a fallback method is needed to communicate the icon's purpose/content. Please see the following examples for cases:
